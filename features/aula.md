# Features
* Achar características de imagem, conceito de Feature
* Reconhecimento de imagem no espaço de busca
## O que são
Detalhes de uma imagem conforme algoritmo, fornecendo informações sobre o que ela significa.  
Exemplos: Pontos de contraste, de forma que caso tenham outros elementos você ainda consegue identificar ela no meio deles.  
* Técnica chamada de Pontos de Interesse ou Key Points: Criamos esses pontos usando o algoritmo.
    * Identifica pixels, vizinhos dos nós, analisa e encontra os tais pontos de interesse.
* SIFT (preciso mas pesado), ORB (mais rápido), SURF, FAST, BRISK (alguns com patente)
    
## Onde usamos?
* Reconhecimento de objetos
* Alinhamento de imagem
* Reconstrução 3d
* Tracking de imagem
* Indexação de imagem
